# SubWorld

## 3D Submarine Game (work in progress)

Written in C ++ it include the following :

- Component Based System in C++
- Steering behaviors with OpenSteer library (http://opensteer.sourceforge.net/)
- Pathfinder using Open Motion Planning Library (OMPL) (https://ompl.kavrakilab.org/) and boost catmull_rom
- AI : Hierarchical Task Networks (HTN) Planner 
- AI : Goal Oriented Action Planning (GOAP) Planner 
- AI :  forward chaining inference engine using CLIPS (https://en.wikipedia.org/wiki/CLIPS)
- 3D : Unigine engine : https://unigine.com/

Note: Because of its size, the Game World is not included in this repository.
